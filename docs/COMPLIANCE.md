# 合规清单 | Compliance Checklist

## 华为应用市场上架合规检查

### 个人开发者上架说明

**可以！个人开发者可以在华为应用市场上架应用。**

华为应用市场支持两种开发者类型：
1. **个人开发者** - 使用身份证实名认证
2. 企业开发者 - 使用营业执照认证

个人开发者上架流程：

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 注册华为开发者账号 | https://developer.huawei.com |
| 2 | 实名认证 | 上传身份证正反面 + 人脸识别 |
| 3 | 签署开发者协议 | 在线签署 |
| 4 | 创建应用 | 填写应用信息 |
| 5 | 上传 HAP | 上传签名后的安装包 |
| 6 | 提交审核 | 等待华为审核（通常 1-3 个工作日） |
| 7 | 上架发布 | 审核通过后自动上架 |

个人开发者 vs 企业开发者：

| 项目 | 个人开发者 | 企业开发者 |
|------|-----------|-----------|
| 认证材料 | 身份证 | 营业执照 |
| 认证时间 | 1-2 天 | 3-5 天 |
| 上架费用 | 免费 | 免费 |
| 应用数量 | 不限 | 不限 |
| 付费应用 | 支持 | 支持 |
| 知识产权 | 个人名义 | 企业名义 |
| 商标注册 | 可选 | 建议 |

注意事项：
- 个人开发者以个人名义享有知识产权
- 专利申请可以以个人名义申请
- 软件著作权可以以个人名义登记
- 商标注册可以以个人名义申请（需有个体工商户执照）
- 华为应用市场对个人和企业开发者的审核标准一致

## 个人知识产权保护指南

### 软件著作权登记

软件著作权是最基础的知识产权保护，建议优先申请：

| 项目 | 说明 |
|------|------|
| 申请渠道 | 中国版权保护中心 https://www.ccpr.com.cn |
| 申请费用 | 免费（2024年起免收费用） |
| 所需材料 | 源代码（前后各30页）、说明书、申请表 |
| 审核周期 | 30-60 个工作日 |
| 保护期限 | 50 年（个人） |
| 申请人 | 个人（身份证） |

### 商标注册

| 项目 | 说明 |
|------|------|
| 申请渠道 | 国家知识产权局 https://sbj.cnipa.gov.cn |
| 申请费用 | 300 元/类 |
| 所需材料 | 商标图样、身份证、个体工商户执照 |
| 审核周期 | 9-12 个月 |
| 保护期限 | 10 年（可续展） |
| 注册类别 | 第9类（软件）、第42类（SaaS） |

注意：个人申请商标需有个体工商户营业执照。

### 专利申请

| 项目 | 说明 |
|------|------|
| 申请渠道 | 国家知识产权局 |
| 申请费用 | 发明专利 900 元，实用新型 500 元 |
| 所需材料 | 请求书、说明书、权利要求书 |
| 审核周期 | 发明 18-36 月，实用新型 6-12 月 |
| 保护期限 | 发明 20 年，实用新型 10 年 |
