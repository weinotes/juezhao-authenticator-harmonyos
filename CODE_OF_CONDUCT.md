# 行为准则

## 我们的承诺

为了营造一个开放、友好、包容的社区环境，我们承诺让所有参与者都能得到尊重和重视。无论年龄、体型、残疾、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、信仰、性别认同和表达、身份和性取向，我们都将一视同仁。

## 我们的标准

### 鼓励的行为

- 使用欢迎和包容的语言
- 尊重不同的观点和经历
- 优雅地接受建设性批评
- 关注社区整体利益
- 对其他社区成员表达同理心

### 不可接受的行为

- 使用带有性暗示的语言或图像，以及任何形式的性关注或性骚扰
- 发表煽动性的评论，包括种族、民族、国籍、性别、性别认同或表达、性取向、残疾、精神状态等方面的偏见
- 任何形式的公开或私下骚扰
- 未经明确许可发布他人的私人信息，如地址或电子邮箱
- 在专业场合被合理认为是侮辱、贬低或歧视的行为

### 我们的责任

项目维护者有责任明确标准的可接受行为，并应针对任何不可接受的行为采取适当和公平的纠正措施。

项目维护者有权删除、编辑或拒绝不符合本行为准则的评论、提交、代码、Wiki 编辑、问题和其他贡献，或者临时或永久禁止任何贡献者从事他们认为不合适的行为。

## 适用范围

本行为准则适用于本项目及其公共空间的每一个场所。当个人代表本项目或其社区时，其代表行为也适用。

## 执法

如发现辱骂、骚扰或其他不可接受的行为，请通过以下方式联系项目团队：contact@juezhaotrade.com

所有投诉都将被审查和调查，并给出必要的回应。项目团队有义务对事件报告者保密。

## 处罚指南

1. **首次警告**：口头或书面警告，清楚地说明不可接受的行为
2. **再次违规**：要求停止任何公开活动或参与项目相关讨论
3. **严重违规**：临时或永久禁止参与项目所有活动
4. **极端情况**：如果行为涉及法律问题，项目团队有权联系相关执法部门

## 处理程序

1. 任何人可以报告违规行为
2. 项目维护者将审查报告，联系报告者获取更多信息
3. 根据情况决定采取何种措施
4. 维护者做出的决定可以在 14 天内向项目负责人提出申诉
